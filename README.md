# Гармония — лендинг салона красоты

Одностраничный сайт-визитка для салона красоты «Гармония»: услуги, галерея работ, отзывы и онлайн-запись.

## Возможности

- Адаптивная вёрстка для десктопа и мобильных устройств
- Фиксированная навигация с эффектом при прокрутке и плавным скроллом к секциям
- Секция услуг: стрижки, окрашивание, маникюр и педикюр, брови и ресницы, уходовые процедуры, укладки
- Блок «О салоне», галерея работ и отзывы клиентов
- Форма онлайн-записи с автоформатированием телефонного номера
- Анимации появления элементов при прокрутке (IntersectionObserver)
- Подключение Google Fonts (Cormorant Garamond, Montserrat)

## Стек

- HTML5
- CSS3 (CSS custom properties, Flexbox, Grid, анимации)
- Vanilla JavaScript

## Структура

- `index.html` — весь лендинг (разметка, стили и скрипты в одном файле)
- `screenshot-full.png`, `screenshot-hero.png` — превью страницы

## Запуск

Отдельная сборка не требуется — это статическая страница.

```bash
# открыть напрямую в браузере
open index.html

# или поднять локальный сервер
python3 -m http.server 8000
# затем открыть http://localhost:8000
```
